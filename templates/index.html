Todo List API

An API that performs all the features of a todo app

Built on the Django and uses the Django Rest Framework.

Features

  • Registerand login a User
  • Create multiple todos
  • Create a todo
  • list all todos
  • Update a todo
  • Delete a todo
  • Filter,search and order todos
  • View Documentation on ReDoc
  • View Documentation on SwaggerUI
